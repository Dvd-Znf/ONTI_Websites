ONTI_Websites

Cum pot Contribuii?
Contributile se pot face cum se fac normal pe GitHub:
Fork --> Change --> Pull Request

Repertoriul principal il puteti gasi aici

El este structurat in doua branch-uri:
Master este unde trebuie sa se faca schimbari
Live este branch-ul din care se serveste site-ul (Nu atinge sau voi ignora PR-ul!)

Puteti sa:
Adaugati noi subiecte din anii precedenti(Vezi folder-ul subiecte)
Sau
Adaugati noi website-uri in folderele unde trebuie (Clasa_(NR_CLASA)/(NR_AN)/.)
Pentru orice alta sugestie/idee va rog deschideti un issue <3
